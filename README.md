# Real-Time Voting Data Engineering Project

## Overview
This project is a real-time data engineering pipeline that simulates and processes voting data. It demonstrates how streaming data systems work using modern big data tools and technologies.

The system collects voting data, processes it in real-time, stores it in a database, and visualizes results using a dashboard.

---

## Architecture

The pipeline includes:

- Data Generation (Voting simulation)
- Stream Processing (Apache Spark Streaming)
- Message Handling (Kafka / streaming logic)
- Storage (PostgreSQL)
- Visualization (Streamlit Dashboard)
- Containerization (Docker)

---

## Tech Stack

- Python
- Apache Spark Streaming
- PostgreSQL
- Streamlit
- Docker & Docker Compose
- JDBC (PostgreSQL connector)
